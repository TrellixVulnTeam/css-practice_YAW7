# Mail 알림 Modal 구현
> 메일이 왔을 때, 알림에 의해 보여지는 간단한 Modal 구현

### **1. 사용스택**
HTML CSS Javascript
<br/>

### **2. 기능**
빨간 버튼을 누르면, 모달창이 사라지는 모습 구현
<br/>


### **3. 코드 특징**
기본적인 CSS의 flex, position:absolute;를 이용하고, JS의 querySelector와 style 변경 동작
<br/>


### **4. 앞으로 추가하면 좋을 기능**
a. 다른 브라우저 화면에서 알림이 왔을 때, 기존의 화면은 opacity가 낮아지고, 모달에 시선이 집중되도록 동적으로 구현
<br/>
